## ğŸ“– Tentang

**HealthMind** adalah platform berbasis web yang dirancang untuk menjembatani kesenjangan akses kesehatan mental. Aplikasi ini menghubungkan pasien dengan psikolog atau dokter spesialis secara mudah, aman, dan efisien.

Dilengkapi dengan dashboard admin yang komprehensif, HealthMind memastikan manajemen pengguna, verifikasi dokter, dan konten edukasi berjalan lancar.
